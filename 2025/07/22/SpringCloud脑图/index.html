July 22, 2025 1423

🧠 Spring Cloud 脑图总览
1️⃣ 服务注册与发现（Service Discovery）
Eureka（已被弃用）

Consul

Nacos

Zookeeper

2️⃣ 配置中心（Config）
Spring Cloud Config

本地 Git / 远程 Git / SVN

Nacos 配置管理

Consul Key/Value Store

3️⃣ 服务调用（Client）
RestTemplate（已逐步淘汰）

OpenFeign

声明式 REST 客户端

支持负载均衡（与 Ribbon 整合 / 与 LoadBalancer 整合）

4️⃣ 负载均衡（Load Balancer）
Ribbon（已被弃用）

Spring Cloud LoadBalancer（替代 Ribbon）

Nginx/Gateway 层转发负载均衡（服务外部）

5️⃣ 服务网关（API Gateway）
Zuul（已弃用）

Spring Cloud Gateway（主流）

路由转发

断言工厂

过滤器工厂

限流、熔断、鉴权

6️⃣ 熔断与限流（Fault Tolerance）
Hystrix（已弃用）

Resilience4j（主流）

熔断器

限流器

重试机制

Sentinel（阿里开源）

7️⃣ 配置与消息总线（Bus）
Spring Cloud Bus

搭配 RabbitMQ / Kafka 实现配置变更广播

Nacos 动态刷新

8️⃣ 分布式链路追踪（Tracing）
Sleuth（结合 Zipkin）

SkyWalking（推荐）

Pinpoint

Jaeger（结合 OpenTracing）

9️⃣ 消息驱动架构（Stream）
Spring Cloud Stream

绑定 Kafka / RabbitMQ

@StreamListener / Function

RocketMQ Stream 支持

🔟 注册中心 UI / 运维工具
Spring Boot Admin

Nacos Console

Consul Web UI

Zipkin UI / SkyWalking UI

🔁 分布式事务（Transaction）
Seata

LCN / TCC / Saga 模式支持

🔒 安全与鉴权（Security）
Spring Cloud Security

OAuth2 + JWT

Spring Authorization Server（替代旧版 OAuth2）

🔄 微服务治理
Gray Release（灰度发布）

Service Mesh：Istio + Envoy

动态路由、版本控制（Header / Metadata）

✅ 典型微服务组合
功能 推荐组件
服务注册发现 Nacos
配置中心 Nacos Config
服务调用 OpenFeign
负载均衡 Spring Cloud LoadBalancer
网关 Spring Cloud Gateway
熔断限流 Sentinel / Resilience4j
配置刷新 Spring Cloud Bus / Nacos
链路追踪 SkyWalking
消息驱动 Spring Cloud Stream (Kafka)
分布式事务 Seata
