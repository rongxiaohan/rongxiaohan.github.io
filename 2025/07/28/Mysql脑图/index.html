MySQL 全面脑图结构

一、基础概念

  • 数据库 vs 数据表

  • SQL 分类

    • DDL（建库建表）

    • DML（增删改）

    • DQL（查询）

    • DCL（授权等）

二、数据类型

  • 数值类型（INT、BIGINT、DECIMAL…）

  • 字符串类型（CHAR、VARCHAR、TEXT…）

  • 时间类型（DATE、DATETIME、TIMESTAMP）

三、数据库对象

  • 表（Table）

  • 视图（View）

  • 索引（Index）

  • 存储过程（Stored Procedure）

  • 触发器（Trigger）

  • 函数（Function）

  • 事件（Event）

四、索引

  • 类型：

    • 普通索引

    • 唯一索引

    • 组合索引

    • 全文索引

    • 前缀索引

  • 数据结构：

    • B+ Tree（主流）

    • Hash（Memory 引擎）

  • 覆盖索引

  • 最左前缀原则

  • 索引失效原因

五、SQL 优化

  • 查询优化：

    • EXPLAIN 分析执行计划

    • 使用索引

    • 避免 SELECT *

    • LIMIT 优化

  • 慢查询分析

  • SQL 语句重写技巧

六、事务（Transaction）

  • ACID 特性

  • 隔离级别：

    • READ UNCOMMITTED

    • READ COMMITTED

    • REPEATABLE READ（默认）

    • SERIALIZABLE

  • 并发问题：

    • 脏读、不可重复读、幻读

  • 锁机制：

    • 表锁 vs 行锁

    • 意向锁、间隙锁、临键锁

七、存储引擎

  • InnoDB（默认）

    • 支持事务、行级锁、外键

  • MyISAM

    • 读写快，不支持事务

  • MEMORY

  • ARCHIVE

八、日志系统

  • binlog（二进制日志，主从复制使用）

  • redo log（InnoDB，崩溃恢复）

  • undo log（事务回滚）

  • slow log（慢查询日志）

  • error log

九、主从复制与高可用

  • 主从复制（基于 binlog）

  • 半同步/异步复制

  • GTID 模式

  • 主主复制

  • MHA、MySQL Router、ProxySQL

十、分库分表

  • 垂直拆分

  • 水平分表

  • 中间件：

    • Sharding-JDBC

    • MyCat

  • 分布式事务（两阶段、最终一致性）

十一、性能调优

  • 参数调优（如 innodb_buffer_pool_size）

  • 缓存机制（Buffer Pool）

  • 查询缓存（8.0已弃用）

  • 索引优化

  • Schema 设计优化

十二、安全与权限

  • 用户与权限管理

  • 最小权限原则

  • SQL 注入防护

十三、常见工具

  • Navicat / DataGrip

  • mysqldump / mysqlbinlog

  • pt-query-digest

  • MySQL Workbench

  • MySQL 5.6：GTID、延迟复制
