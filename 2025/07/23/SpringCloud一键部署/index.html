💡 Step 3：Jenkins 安装插件 + 准备环境

安装 Jenkins 插件：

  • ✅ Git Plugin

  • ✅ Maven Integration Plugin

  • ✅ Docker Pipeline Plugin

配置 Jenkins：

  • 系统配置中添加 Docker 命令路径

  • 添加 Maven 全局设置

  • 配置 Docker 仓库登录凭证（Harbor 或 DockerHub）

  • SSH 登录目标服务器（部署机器）

💡 Step 5：部署结果验证

  • Jenkins 构建成功后访问：http://部署服务器IP:8081/

  • Docker 查看容器是否启动成功：docker ps

  • 镜像是否被推送成功：docker images

✅ 技术总结

技术 作用
Spring Cloud 微服务架构
Jenkins 持续集成、持续部署
Docker 容器化部署
Harbor / DockerHub 镜像仓库
SSH Plugin / Docker CLI 远程部署

🔒 安全建议

  • 镜像仓库使用 Harbor + 登录凭证

  • Jenkins 主机不要暴露给公网

  • Docker 容器启动参数使用资源限制 --memory --cpus
