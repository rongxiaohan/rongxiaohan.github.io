省份数量

July 15, 2025 3464

题目描述：

有 n 个城市，其中一些城市彼此相连，另一些没有相连。如果城市 A 和城市 B 直接或间接相连（通过其他城市间接连接），它们属于同一个省份。

给定一个 n x n 的邻接矩阵 isConnected，其中 isConnected[i][j] = 1 表示城市 i 与城市 j 直接相连。

返回省份的数量。

✍️ 示例：

输入：

输出：2
解释：城市 0 和城市 1 构成一个省份，城市 2 单独一个省份。

解法一：DFS 深度优先搜索

将图看作无向图的邻接矩阵，遍历每个城市，对未访问的城市进行 DFS，搜索能连通的所有城市，每进行一次 DFS 就是一整个省份。

解法二：并查集（Union Find）

每个城市初始化为独立集合，遍历所有连接关系，将能连通的城市合并为一个集合，最后统计集合数量即可。

解法三：BFS 广度优先搜索

和 DFS 思路一样，只是用队列代替递归。

总结：

方法 时间复杂度 空间复杂度 适合场景
DFS O(n²) O(n) 实现简单，适合图稠密
BFS O(n²) O(n) 广度搜索稳定
并查集 O(n²) + α(n) O(n) 多次查询合并，性能优
